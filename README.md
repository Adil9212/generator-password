🔐 MySecurePass — Генератор & Анализатор Стойкости Паролей

Современное, быстрое и полностью конфиденциальное веб-приложение для создания криптостойких паролей и мгновенного анализа их надежности.

Инструмент сочетает в себе гибкий генератор на основе настраиваемых пулов символов и продвинутый эвристический анализатор сложности, работающий полностью на стороне клиента.

✨ Ключевые особенности

🛡️ Умный Генератор Паролей

Гарантия сложности: Алгоритм гарантирует наличие как минимум одного символа из каждого выбранного набора (регистр, цифры, спецсимволы), предотвращая генерацию «слабых» комбинаций.

Защита от двусмысленности: Возможность исключения визуально похожих символов (таких как i, l, 1, o, 0, O) для предотвращения ошибок при ручном вводе.

История генераций: Локальное сохранение последних созданных паролей для быстрого переключения.

📊 Продвинутый Анализатор (Аналог zxcvbn)

Расчет энтропии: Вычисление информационной плотности пароля в битах по формуле Шеннона.

Эвристический анализ уязвимостей: Детектирование популярных паролей из встроенной базы данных, выявление клавиатурных последовательностей (например, qwerty или 12345) и повторяющихся символов.

Оценка времени взлома: Динамический расчет времени, необходимого для перебора пароля современным хакерским оборудованием (Brute Force).

🔒 Безопасность и Приватность

100% Client-Side: Все вычисления и проверки происходят локально в браузере. Ваши пароли никогда не отправляются на сервер, обеспечивая абсолютную защиту персональных данных.

🛠️ Технологический стек

Frontend: React 18, JavaScript (ES6+)

Стилизация: Tailwind CSS

Сборка проекта: Vite

🚀 Быстрый старт (Локальный запуск)

Для запуска проекта на вашем компьютере выполните следующие шаги:

Клонируйте репозиторий:

" git clone [https://github.com/Adil9212/generator-password.git](https://github.com/Adil9212/generator-password.git) "


Перейдите в папку проекта:

" cd password-generator-tool "


Установите зависимости:

" npm install "


Запустите сервер для разработки:

" npm run dev "


Откройте в браузере адрес http://localhost:5173.
